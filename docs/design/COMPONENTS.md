## Component Reuse Guidelines

1. **Consistency:** Always use these defined components
2. **Variants:** Use props for variants, not new components
3. **Composition:** Build complex UI from simple components
4. **States:** Every interactive component needs all states defined
5. **Accessibility:** Include focus states, ARIA labels
6. **Animation:** Subtle, 150ms default, ease-in-out
